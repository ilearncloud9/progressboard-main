## The App

### JSON Rest API

The app provides raw json data via the routes under /api/v1

- /api/v1/data
- /api/v1/user_repos
- /api/v1/users

### The HTML views

- / (Heatmap View grouped by user and session)
- /updates (Timeline Grid View sorted chronologically)
